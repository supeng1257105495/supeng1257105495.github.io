CSS3新特性

发表于 2019-08-22

css新特性

过渡

  1. 语法：transition： CSS属性，花费时间，效果曲线(默认ease)，延迟时间(默认0)
    语法2： transition-property: width;
    transition-duration: 1s;
transition-timing-function: linear;
transition-delay: 2s;
  2. 举例1：transition：width,.5s,ease,.2s //宽度从原始值到制定值的一个过渡，运动曲线ease,运动时间0.5秒，0.2秒后执行过渡
  3. 举例2：transition：all,.5s //所有属性从原始值到制定值的一个过渡，运动曲线ease,运动时间0.5秒

动画

  1. 语法1：animation：动画名称，一个周期花费时间，运动曲线（默认ease），动画延迟（默认0），播放次数（默认1），是否反向播放动画（默认normal），是否暂停 动画（默认running）
  2. 举例：animation: logo2-line 2s linear;//执行一次logo2-line动画，运动时间2秒，运动曲线为 linear;
    animation: logo2-line 2s linear 2s;//2秒后开始执行一次logo2-line动画，运动时间2秒，运动曲线为 linea;
animation: logo2-line 2s linear alternate infinite;//无限执行logo2-line动画，每次运动时间2秒，运动曲线为 linear，并且执行反向动画
  3. ps: animation-fill-mode : none | forwards | backwards | both;//none：不改变默认行为。 
    forwards ：当动画完成后，保持最后一个属性值（在最后一个关键帧中定义）。    
backwards：在 animation-delay 所指定的一段时间内，在动画显示之前，应用开始属性值（在第一个关键帧中定义）。 
both：向前和向后填充模式都被应用。

阴影

  1. 语法：box-shadow: 水平阴影的位置 垂直阴影的位置 模糊距离 阴影的大小 阴影的颜色 阴影开始方向（默认是从里往外，设置inset就是从外往里）;

边框图片

  1. 语法：border-image: 图片url 图像边界向内偏移 图像边界的宽度(默认为边框的宽度) 用于指定在边框外部绘制偏移的量（默认0） 铺满方式–重复（repeat）、拉 伸（stretch）或铺满（round）（默认：拉伸（stretch））;

倒影

  1. 语法：-webkit-box-reflect:方向[ above-上 | below-下 | right-右 | left-左 ]，偏移量，遮罩图片

  2. 举例:

    下倒影

    <p class="reflect-bottom-p"><img src="test.jpg" class="reflect-bottom"></p>
<style>
.reflect-bottom-p {
       padding-bottom: 300px;
   }

.reflect-bottom {
       -webkit-box-reflect: below;
   }   
</style>

盒模型定义

  1. 为什么是盒模型？

    • 网页由一个个盒子组成的原因是：每个HTML元素在页面上加载的时候，都是一个盒子。只是默认情况下，盒子的边框隐藏、背景透明，所以我们只看到了盒子里面的内容。也就是说，一个盒子就是一个元素，一个元素就是一个盒子。

  2. 盒子有哪些属性？

    • 边框：相当于生活中的纸盒子；它有样式border-style（虚线、实线等）、颜色border-color、粗细border-width、阴影border-dashed

    • 内边距：内容与边框之间的距离（内容就是生活中你装在盒子里的东西，如衣服），有上下左右边距之分

    • 外边距：盒子与盒子之间的距离（想像一下，两箱子衣服放在地板上），有上下左右之分。上下边距会相互叠加

    • 背景层：background属性。想像一下盒子是透明的，是有图案的。

  3. 盒子的大小？

    • 不设置盒子的width属性：block元素的内容会填充满父元素。增加了边距后，内容宽度被挤压

    • 设置了width属性：width只代表了内容的宽度，添加的边距会在width的基础上叠加，使得盒子的大小扩展
