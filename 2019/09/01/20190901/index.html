发表于 2019-09-01

2019后半年规划几个目标

  1. 理财 对自己的收入支出进行规划管控 开始进行对于2020年的计划开始存款。
  2. 时间 规划每天的时间给自己一个合理的时间规划 。
  3. 学习 在合理的时间规划中完成学习保证基础的学习时间不会被浪费。
  4. 锻炼 每天抽出40-80分钟进行锻炼，好的身体才是革命的本钱。
  5. 目标 为了短期目标 保持、坚持自己的规划。

小结：订制下的目标按时完成 养成好的习惯 培养自己的副业 对自己的规划有planB 剩下的交给坚持努力。頑張ります。
